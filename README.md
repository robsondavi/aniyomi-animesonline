# Repositório Aniyomi - AnimesOnline.cloud

Repositório de extensão personalizada do Aniyomi para o site AnimesOnline.cloud

## Como usar no Aniyomi

1. Abra o **Aniyomi**
2. Vá em **Configurações → Extensões de Anime → Repositórios**
3. Adicione a URL abaixo:

```
https://SEU-USUARIO.github.io/SEU-REPOSITORIO/index.min.json
```
